1. Introduction

Dose-response meta-analysis combines evidence from multiple studies to estimate the relationship between exposure levels (dose) and outcomes. This is particularly valuable when individual studies examine different dose ranges or categories.

Key Features

  • Multiple model types: Linear, quadratic, splines, fractional polynomials, and more
  • Covariance handling: Greenland-Longnecker method for correlated estimates
  • Missing data: Hamling reconstruction for incomplete reporting
  • Model comparison: AIC, BIC, and likelihood ratio tests
  • Bayesian inference: MCMC-based estimation with credible intervals
  • Publication-quality plots: D3.js visualization with confidence bands

2. Data Format

Dose-response data should be structured with one row per dose level per study:

const data = [
    // Study 1: Coffee and all-cause mortality
    { study: 'Study A', dose: 0,   yi: 0,     vi: 0,      n: 1000, cases: 50 },
    { study: 'Study A', dose: 1,   yi: -0.05, vi: 0.01,   n: 1200, cases: 55 },
    { study: 'Study A', dose: 3,   yi: -0.12, vi: 0.015,  n: 800,  cases: 30 },
    { study: 'Study A', dose: 5,   yi: -0.08, vi: 0.02,   n: 500,  cases: 22 },
    // Study 2
    { study: 'Study B', dose: 0,   yi: 0,     vi: 0,      n: 2000, cases: 100 },
    { study: 'Study B', dose: 2,   yi: -0.10, vi: 0.008,  n: 1800, cases: 80 },
    // ... more studies
];
Note: The reference category (dose=0) should have yi=0 and vi=0. Effect sizes (yi) are typically log relative risks or log odds ratios.

Required Fields

FieldDescriptionRequired
studyStudy identifierYes
doseExposure levelYes
yiEffect size (log scale)Yes
viVariance of effect sizeYes
nSample size / person-yearsFor covariance
casesNumber of eventsFor covariance

3. Available Models

Linear

y = β × dose

Simplest model, constant effect per unit dose

Quadratic

y = β₁×dose + β₂×dose²

Captures U-shaped or inverted-U relationships

Log-Linear

y = β × log(dose + c)

Diminishing returns at higher doses

Restricted Cubic Splines

Flexible smooth curves with knots

Most flexible for complex relationships

Piecewise Linear

Linear segments with breakpoints

Threshold effects or changing slopes

Emax

y = E_max × dose / (ED₅₀ + dose)

Pharmacological dose-response

4-Parameter Sigmoid

Hill equation with min/max

S-shaped curves with plateau

Fractional Polynomials

Powers from {-2,-1,-0.5,0,0.5,1,2,3}

Flexible parametric curves

4. Basic Analysis

Running a Linear Dose-Response Analysis

// Run linear dose-response meta-analysis
const result = AdvancedMethods.doseResponseMA(data, 'linear', 'REML');

// Access results
console.log('Slope:', result.coefficients[0].estimate);
console.log('95% CI:', result.coefficients[0].ci95);
console.log('Heterogeneity I²:', result.I2.toFixed(1) + '%');
console.log('Tau²:', result.tau2.toFixed(4));

5. Model Comparison

            
Compare multiple models to find the best fit using AIC, BIC, and Akaike weights:


            
// Compare multiple dose-response models
const comparison = AdvancedMethods.compareDoseResponseModels(data, 'REML');

// Results include:
// - AIC and BIC for each model
// - Akaike weights (probability each model is best)
// - Best model recommendation
// - Likelihood ratio tests vs linear model

console.log('Best model:', comparison.bestModel);
comparison.models.forEach(m => {
    console.log(`${m.model}: AIC=${m.aic.toFixed(1)}, weight=${(m.weight*100).toFixed(1)}%`);
});


            
            

                

            

        


        
        

            
6. Advanced Models


            
Restricted Cubic Splines

            
// Spline with 3 knots (default: 10th, 50th, 90th percentiles)
const splineResult = AdvancedMethods.doseResponseMA(data, 'spline', 'REML', {
    knots: 3  // or specify exact positions: knots: [1, 3, 5]
});

// Spline coefficients represent the shape parameters
splineResult.coefficients.forEach((c, i) => {
    console.log(`Spline coefficient ${i}: ${c.estimate.toFixed(4)}`);
});


            
Piecewise Linear (Segmented Regression)

            
// Auto-detect optimal breakpoint(s) using AIC
const piecewise = AdvancedMethods.piecewiseLinearDR(data, 'REML');

// Or specify breakpoints manually
const piecewise2 = AdvancedMethods.piecewiseLinearDR(data, 'REML', [2.5]);

// Results include segment-specific slopes
console.log('Segments:', piecewise.segments);
// e.g., "Segment 1 (0-2.5): slope = -0.08"
//       "Segment 2 (2.5+): slope = -0.02"


            
Fractional Polynomials

            
// First-degree fractional polynomial (best single power)
const fp1 = AdvancedMethods.fractionalPolynomialDR(data, 'REML', 1);
console.log('Best power:', fp1.powers);  // e.g., [0.5] for sqrt(dose)

// Second-degree fractional polynomial (two powers)
const fp2 = AdvancedMethods.fractionalPolynomialDR(data, 'REML', 2);
console.log('Best powers:', fp2.powers);  // e.g., [-1, 2]


            
            

                

            

        


        
        

            
7. Bayesian Analysis

            
Bayesian dose-response provides posterior distributions, credible intervals, and probability statements:


            
// Run Bayesian dose-response meta-analysis
const bayesResult = BayesianMA.bayesianDoseResponse(data, {
    model: 'linear',      // 'linear', 'quadratic', or 'spline'
    iterations: 10000,    // MCMC iterations
    burnin: 2000,         // Burn-in period
    chains: 4,            // Number of chains
    priorBeta: { mu: 0, sigma: 10 },     // Prior on coefficients
    priorTau: { type: 'halfCauchy', scale: 0.5 }  // Prior on heterogeneity
});

// Posterior summaries
bayesResult.coefficients.forEach(c => {
    console.log(`${c.name}: mean=${c.mean.toFixed(4)}`);
    console.log(`  95% CrI: [${c.ci95.lower.toFixed(4)}, ${c.ci95.upper.toFixed(4)}]`);
    console.log(`  P(slope < 0): ${(c.probNegative * 100).toFixed(1)}%`);
});

// Convergence diagnostics
console.log('Rhat:', bayesResult.diagnostics.beta[0].rhat);  // Should be < 1.1
console.log('ESS:', bayesResult.diagnostics.beta[0].ess);    // Should be > 400


            

                Important: Always check convergence diagnostics. Rhat > 1.1 or ESS < 100
                indicates the MCMC chains may not have converged. Increase iterations if needed.
